## 💡 核心功能

✅ **夜间模式**：支持夜间模式，保护眼睛 👀 人人有责。

✅ **缓存与观影进度**：支持视频缓存，并自动保存当前播放进度，确保下次打开时能够继续观看。

⬜ **多设备云同步**：基于各种第三方同步工具或协议，确保书源、播放进度和源规则等同步到云端，多设备共享。
